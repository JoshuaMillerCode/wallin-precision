# Launch SEO Local Business

## Primary Goal
Help a local training business get discovered by nearby customers while keeping pages useful for real humans.

Known project context:
- business name: `Wallin Precision`
- currently visible location: `Unionville, NC`
- currently visible offerings: firearms training and concealed carry training
- concealed carry training is the confirmed primary offer
- current online presence is mostly Facebook-first, so the website should become the primary source of truth

## Priorities
- clear service pages
- location-aware content
- strong page titles and descriptions
- consistent internal linking
- FAQ and trust content that answers real questions

## Core Recommendations
- Create a focused home page with clear location/service intent.
- Create individual course or service pages when intent differs.
- If multiple cities are targeted, only create location pages that have meaningful custom content.
- Use descriptive slugs and headings.
- Make phone, contact, and booking paths obvious.
- Create a dedicated concealed carry class page as a priority page.
- Preserve `Unionville, NC` and nearby service-area intent in homepage, metadata, and contact copy.
- Make sure key metadata and internal linking reinforce concealed carry intent without stuffing.

## Metadata Pattern
For each key page, define:
- title with service plus location intent
- concise meta description with value and CTA
- open graph title and description
- canonical URL

## Content Structure
For local landing pages, include:
1. what the business offers
2. who it serves
3. location or service area context
4. trust signals
5. FAQ
6. booking CTA

Suggested early page set for this project:
- home
- concealed carry class
- beginner firearms training
- advanced or private training
- about/instructor
- contact
- FAQ

Priority internal links:
- home -> concealed carry class
- concealed carry class -> contact
- concealed carry class -> beginner or advanced training

## SEO Guardrails
- Avoid thin location pages.
- Avoid keyword stuffing.
- Do not duplicate the same copy across course pages.
- Keep structured data accurate and limited to what is actually true.

## Output Pattern
When asked for SEO help, provide:
1. target search intent
2. recommended pages
3. metadata suggestions
4. internal linking suggestions
5. content gaps
